02.01.05: treatment

On this project we learned how to make a treatment, Which is basically writing down all the scenes before you shoot them. We understood that it is very important to write down the script first and then to act.By doing this we saved ourselves from getting stuck and not knowing what to do. Also the treatment helped everyone in our team to be in the same page and understand what we have to do. We had no difficulties with the treatment
